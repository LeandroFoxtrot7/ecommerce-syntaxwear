#  SyntaxeWear - E-commerce de Tênis

Um site de e-commerce moderno para venda de tênis e sneakers online, desenvolvido com HTML, CSS e estrutura responsiva.

##  Sobre o Projeto

**SyntaxeWear** é uma plataforma de e-commerce especializada em tênis e sneakers premium. O projeto apresenta uma interface limpa e moderna, focada na melhor experiência do usuário.

##  Estrutura do Projeto

```
ecommerce-syntaxwear/
├── index.html                 # Página principal
├── README.md                  # Este arquivo
├── css/
│   ├── reset.css             # Reset de estilos CSS
│   ├── variables.css         # Variáveis CSS (cores, tipografia, espaçamentos)
│   ├── base.css              # Estilos base e globais
│   └── components/
│       ├── header.css            # Estilos do cabeçalho
│       ├── hero.css              # Estilos da seção hero
│       ├── product-category.css  # Estilos das categorias
│       ├── product-grid.css      # Estilos da grade de produtos
│       |__ footer.css            # Estilos do rodapé
│   
└── images/
    ├── banners/              # Banners do site
    ├── favicons/             # Ícones do navegador
    ├── icons/                # Ícones gerais (menu, usuário, etc)
    ├── logo/                 # Logo da marca
    └── products/             # Imagens dos produtos
```

##  Componentes Principais

### Header
Navegação principal com:
- Logo da marca
- Menu responsivo (hambúrguer)
- Navegação por categorias (Masculino, Feminino, Outlet)
- Ações rápidas (lojas, sobre, usuário, carrinho)

### Hero
Seção de destaque com banner promocional

### Categorias
Exibição de categorias de produtos:
- Masculino
- Feminino
- Outlet
- Nossas lojas
- Sobre

### Grade de Produtos
Layout responsivo para exibição de produtos com imagem e nome

### Footer
Rodapé com:
- Logo e redes sociais
- Links de navegação por categoria
- Informações de contato
- Copyright

##  Recursos CSS

- **Reset.css**: Remove estilos padrão dos navegadores
- **Variables.css**: Variáveis CSS para manter consistência de design
- **Base.css**: Estilos globais e tipografia
- **Components**: Estilos modularizados por componente

##  Responsividade

O projeto utiliza Mobile-first approach com media queries para adaptação em diferentes dispositivos:
- Smartphones
- Tablets
- Desktops

##  Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Estilos e layout responsivo
- **SVG**: Ícones vetoriais escaláveis

##  Imagens

Organização de assets por tipo:
- **Banners**: Imagens promocionais
- **Favicons**: Ícones para abas do navegador
- **Icons**: Ícones da interface (hambúrguer, usuário, carrinho, redes sociais)
- **Logo**: Identidade visual da marca
- **Products**: Catálogo de imagens dos produtos

##  Como Usar
